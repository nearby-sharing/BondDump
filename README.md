# BondDump

Small helper to dump the [`System.Linq.Expressions.Expression`](https://learn.microsoft.com/en-us/dotnet/api/system.linq.expressions.expression) -- generated by [Microsoft Bond](https://microsoft.github.io/bond/) at runtime -- to a C# source file.

This enables trimming (and later AOT compilation) of NearShare apps.
